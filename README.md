# Cachubana

### Demo

Puedes acceder al demo [aca](https://frubanafrontend.omarefg.now.sh/)

### Requisitos
* nodejs
* yarn o npm

### iniciar el proyecto
* clonar el repositorio:

  ``` bash
  $ git clone https://github.com/omarefg/cachubana.git
  ```
* instalar dependencias
  ``` bash
  $ cd cachubana
  $ yarn install
  ```
* iniciar el servidor
  ``` bash
  $ yarn start
  ```
